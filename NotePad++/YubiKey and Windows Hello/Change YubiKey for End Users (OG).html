CFPB Service Desk Guide

Change YubiKey PIN for Windows

  1.   Click the Start menu icon and search for, then choose ‘Settings’'.

  2.   Click ‘Accounts’, then ‘Sign-in options’.

  3.   Click ‘Security Key'.

  4.   Click ‘Security Key’, then ‘Manage’.

  5.   When prompted, use your finger to touch the YubiKey.

  6.   Under Security Key PIN, Select 'Change'to change the PIN number for your YubiKey.

  7.   Enter the default PIN 17001700, then a new 6-8 digit PIN twice.

    Example: Your current PIV pin would be accepted.

  8.   After all fields have been completed, click OK.

Change YubiKey PIN for macOS

  1.   Open YubiKey Manager from the Applications folder.

    Note: If the application is not present, install from Self Service.

  2.   Select ‘FIDO2’ from the Applications menu.

  3.   Click 'Change PIN'.

  4.   Enter your current PIN, then a new PIN twice. Click ‘Change PIN’.

  5.   You should briefly see a message saying “Changed FIDO2 PIN”.
