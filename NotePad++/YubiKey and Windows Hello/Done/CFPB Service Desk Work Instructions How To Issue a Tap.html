CFPB Service Desk Work Instructions:

How to Issue a Temporary Access Pass (TAP)

Create a TAP

  1.   Sign in to the Azure portal by using your Azure AD ADM account.

  2.   Select Azure Active Directory > browse to Users > select the user's profile > choose
    Authentication methods.

      Note: If needed, select the option Try the new user authentication methods experience.

  3.   Select Add authentication methods.

  4.   Below Choose method, select Temporary Access Pass.

  5.   Accept the default time or duration by selecting Add.

      Note: Once added, the details of the TAP are shown.

  6.   Document the TAP value and provide this password to the user direct chat
    via Teams.

      Note: The TAP will only show itself once, you can't view this value after you select OK.
