CFPB ENF Trial Device Setup Instructions

Printers – HP M234 (B&W) and HP M479 (color)

  1. Connecting to a CFPB Laptop

    1. Users should be aware that connection of the HP printer to a CFPB Laptop can only be done via a USB connection

    2. A user should find the associated USB cable, which should have come with the printer and connect it between the user’s CFPB laptop and the printer itself

    3. Users may not connect to the printer via WiFi or a Network, although this capability will be available on the printer, it should be avoided.

      1. Avoid using the HP smart app both the desktop and mobile app for any printing or scanning

      2. Do not connect to the printer via Wi-Fi and only use the HP M234sdw with CFPB

        devices

        1. Mobile printing capabilities are prohibited including

          1. Apple AirPrint

          2. HP Smart App

          3. Mopria

            USB Port

            Power Port

          4. Wi-Fi Direct Printing


  2. Installing the Required Driver(s)

    1. Users should find the Windows button ( icon) located on their CFPB laptop keyboard, and press it

    2. This action will prompt the Start menu to appear from the bottom left of the user’s laptop screen

    3. A user should locate the Software Center icon which may appear to the right of the Start Menu under Productivity or can be searched by starting to type “Software Center”

    4. Select the icon for Software Center to open the application

    5. Users will navigate to Applications found along the left side of the Software Center, and select it

    6. The Application page will open, and the user should look for the HP LaserJet MFP M234sdw/479

      1. Users may see the HP LaserJet MFP M234sdw or 479 software listed within the body of the Application page

      2. Or users may navigate to the upper right of the page and find the Search bar and use it to search for the software to be installed

      3. The user should ensure that the printer is connected to the user’s laptop and powered

        on

        1. This must be confirmed, or the next steps will not work as intended. (HP 234 ONLY, HP 479 drivers can be pre-installed without being connected)

      4. Once the software has been found, the user should first click on the software icon then select the Install button and wait for installation to complete. This should take a few minutes and users should attend the process, not letting the laptop close, lose power or fall asleep.

      5. When the installation is complete, the user will see that the HP LaserJet MFP M234sdw/479 status is Installed

*HP 479 only works for printing via this method. Please reach out to a user with the installed software necessary for scanning or contact Rafael Samonte for assistance

Mi-Fi Connection Guide

  1. Power on the mi-fi by holding down the power button located on the top of the device. Hold the power button down until graphics are seen on the screen then release.

  2. Once the device is powered on, you will need to connect your laptop to it. (Windows 11 go to Step 3 after doing this. Windows 10, do step 2a before continuing to step 3).

    2a. For Windows 10 machines, locate the chevron icon ^. It will be on the bottom right hand side. Click it to open up a sub menu with icons in it.


  3. Click the globe icon to bring up your wi-fi connections menu. . This will bring up a list of available wireless connections.

  4. Locate the name of the mi-fi that you will be connecting to in this menu 4a. For the AT&T MI-FI, it will be called RG2102_5C9A

    4b. For the Verizon MI-FI, it will be called Verizon-MiFi8800L-0404

  5. Click on the Mi-Fi name. It will bring up a menu prompting you to enter a password. 5a. UNCHECK the box that says “Connect by Default”.

  6. Enter the password for the mi-fi you are using 6a. For the AT&T mi-fi, the password is: 4120f9c3

    6b. For the Verizon mi-fi, the password is: 167974e1

  7. Press Connect. Allow the mi-fi to fully connect

  8. Once connected, your VPN will automatically connect to the internet


Printing Guide

1. When printing, use the drop-down menu to select the printer you are using.


1a. For the HP M234 Printer (Black & White), select the option in the drop down that says “HP Laserjet M234sdw”.

1b. For the HP M479fdw Printer (Color w/Scanner), select the option in the drop down that says “HP Color Laserjet Pro M478f-9f UPD PCL 6”
