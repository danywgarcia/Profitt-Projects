CFPB Service Desk Guide:

A QRG Workspace 1700G Printer List August 2023

Overview

The Consumer Financial Protection Bureau (CFPB) has a requirement to....

For Windows Users

  1. Have the user log in to their CFPB laptop/desktop using their preferred method.

  2. Have the user open the Start menu to search for and click on the 'Settings' icon.

  3. Navigate the user to the left side menu and click 'Accounts ', then 'Sign-in options '.

  4. In the Sign-in options window, ask the user to find and click 'Security Key'.

  5. Once expanded, have the user click 'Manage' next to 'Sign in to apps with a physical security key'.

  6. Have the user touch the metal tab (a gold strip on the side or gold circle) on their YubiKey to verify user
    presence when prompted.

  7. Under Security Key PIN, have the user select 'Change' to change the PIN number for their YubiKey.

  8. Have the user enter their current PIN, and then a new PIN, entering it a second time to confirm.

    Note: Remind the user that it is possible to use the same PIN code as their current PIV PIN in case they
    have updated their PIV PIN.

  9. After all fields have been completed, have the user click OK.

For macOS Users

The pin is changed using the YubiKey Manager. If the application is not present, have the user install it
from Self Service.

  1. Have the user open the 'Yubikey Manager' app.

  2. Direct the user to select 'FIDO2' from the Applications menu.

  3. Have the user click 'Change PIN'.

  4. Have the user enter their current PIN, then create a new PIN, entering it a second time to confirm. Click
    'Change PIN'.

    Note: Remind the user that it is possible to use the same PIN code as their current PIV PIN in case they
    have updated their PIV PIN.

    They should briefly see a message saying 'Changed FIDO2 PIN'.
