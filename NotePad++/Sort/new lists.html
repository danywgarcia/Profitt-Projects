The list-style-type Property

To create a numbered list:

  1. Contact the vendor to report the supply need, providing as much detail as possible.
  2. Staff must ensure proper documentation of the issue prior to contacting the appropriate vendor. To do this, open ServiceNow and search for a ticket referring to the issue. If no ticket exists, create a ticket detailing the specific printer/copier, location, issue, the time the incident was observed, the asset tag number, and the printer serial number (available in the CMDB entry for the printer). For information about the appropriate categorization and creation of these tickets, review Appendix C.

To add a second numbered list:

  1. Use the same CSS
  2. This will make it cleaner
