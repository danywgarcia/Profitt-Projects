Adding a New Network Printer

When adding a new printer to the network, the technician must add and configure the printer, and then test the printer for functionality. Finally, create a shortcut on the T:\ drive. This section outlines the complete process to follow when adding a new printer to the CFPB network.

 

Adding the Printer

To add a new printer to the network:

  1. Click the Start button, select All Programs > Accessories, and then select Remote Desktop Connection from the Accessories pop-up menu.

  2.  

  3. On the Remote Desktop Connection window, type “whqprtw02” (for HQ location) into the Computer field, and then click Connect.

     

    Note: Print server names for the regional sites are:

    • CHI-PRT-W-V01 (Midwest) - 10.95.144.11

    • SFR-PRT-W-V01 (West)- 10.95.140.11

    • NYC-PRT-W-V01 (Northeast) - 10.95.142.11

    • WHQPRTW02 (Headquarters) – 10.96.42.32

       

      Note: Please see the Enhanced User Access Request SOP for details on obtaining administrative access.

  4. On the Windows Security window, enter your ADM password into the first field, and then click the OK button.

  5.  

  6. On the CFPB Notice and Consent Banner, you need to read and fully understand the disclaimer. If you agree, click-on the OK button.

  7.  

  8. Click the Start button in the lower left corner of the server desktop, and then select Print Management from the resulting pop-up menu.

  9.  

  10. Click the Yes on the User Account Control dialog box.

  11.  

  12. The Print Management Console window will display. Expand the WHQPRTW02 (local) item.

  13.  

  14. Select the Ports item from the four-item “tree” displayed below it.

  15.  

  16. Choose an IP address that is within the subnet on the floor where you wish to add the new printer, but also one that is not currently being used (write down the applicable IP address). The last octet must be between 20 and 49. If you are unsure or experience issues, contact the Deskside Support Lead.

  17.  

  18. Right-click on Printers in the tree under the WHQPRTW02 (local) item and then select the Add Printer option from the drop-down menu.

  19.  

  20. When the Printer Installation window is displayed, select the Add a TCP/IP or Web Services Printer by IP address or hostname radio button option, and then click the Next button.

  21.  

    22. When the Printer Address window displays, perform the following:

  22.  

  23. Select the TCP/IP Device option from the Type of Device drop-down menu.

  24.  

  25. Type the applicable IP address (do not use underscores) into the Host name or IP address field (this value will be auto-written into the Port name field as well).

  26.  

  27. Click the Next button.

  28.  

    29. The system will search to detect the applicable port. When it does, the Additional port information required window will display.

  29.  

  30. In the Device Type frame, use the default values (Standard radio button option selected and Generic Network Card in the associated text field), and then click the Next button.

  31.  

  32. On the Printer Driver window, select the Use an existing printer driver on the computer radio button option, select Xerox Global Print Driver PCL6 and then click the Next button.

     

    Note: If the applicable driver is not available from the pick lists, perform the following process:

     

    1. Go to the standard support site for the printer being added.

    2. Download the driver for the printer being added.

    3. On the Printer Installation window, click the Have Disks button.

    4. Navigate to the applicable driver file’s location, and then select it.

    5. Click the Next button.

  33.  

  34. On the Printer Name Sharing Settings window, the printer you defined will be displayed in both the Printer Name and Share Name fields.

  35.  

  36. In the Printer Name field, rename the printer in accordance with standard CFPB nomenclature rules (for this example, the printer might be renamed “1700G_6013_XeroxVL”), and then copy and paste that value into the Share Name field.

     

    Note: The standard CFPB format is <1700G>_<Room Number>_<Printer Model>.

  37.  

  38. Click the Next button to display the Printer Found window.

  39.  

  40. Click the Next button to display the final window of the installation wizard to begin installing the driver and printer.

  41.  

  42. Click the Finish button to complete the process.

    • Default Black and White Printing

    All printers should be configured to print in black and white by default.

    1. From the print management console, right-click on the printer and select Properties…

    2.  

    3. Select Preferences.

    4.  

    5. Under Xerox Black and White, select on from the dropdown menu.

       

    6. Select Apply then OK to save settings.

  •  

    • Printing a Test Page and Creating the Printer Shortcut

    Use the following procedure to a print test page and create the shortcut for the new network printer.

    1. After ensuring the new network printer has been properly added, set up and configured, attempt to connect to the printer and send a test page to the printer.

    2.  

    3. After successfully printing a test page, create the new printer shortcut and place it in the appropriate location in the Shortcuts folder on the T:\ drive:

    4.  

    5. Navigate directly to the print server from your computer using the following procedure:

       

      1. Click the Start button.

      2. Type “run” in the search field and press the Enter key.

      3. On the Run window, type “whqprtw02” in the Open field, and then press the Enter key.

         

    6. Locate the network printer you just added in the listing, right-click on it, and then select the Create Shortcut option from the pop-up menu. (The shortcut will be created on your desktop.)

    7.  

    8. Locate the shortcut on your desktop and copy it.

    9.  

    10. Navigate to the Shortcuts folder (T:\Shortcuts\PRINTERS), locate the appropriate subfolder (i.e., the correct location, building and floor), and paste the new shortcut into that subfolder

    • Enabling the Print Separator

    After successfully printing a test page, the print separator page feature must be enabled.

    1. From the print management console, right-click on the printer and select Properties…

    2. Select the Advanced tab, and then select Separator Page…

    3. Select Browse… Select the sysprint SEP file from the list and then click Open.

    4. Click OK to close out all open windows.

    5. Print a test page to the printer to ensure that the separator page prints as expected.

  •  

    • Restricting a printer

    If a printer needs to be restricted for use by only specified users or groups:

    1. From the print management console, right-clickon the printer and select Properties…

    2. Select the Security tab and remove any users or groups that should not have access to the printer from the Group or user names list (for this example, the Everyone group is present) by highlighting the name and clicking Remove.

    3. Select Add. Begin typing the name of the user or group in the Enter the object names to select field, and then click Check Names.

    4. Select the correct user or group from the list of matches. Click OK. The selected user or group will display on the Group or usernames list.

    5. Repeat steps 3 and 4 for any additional users or groups that need to be added.

    6. Click OK to save the changes and close the window.

    • SNMP Configuration for SolarWinds

    1. From the print management console, right-clickon the printer and select Properties…

    2. Select the Ports tab. Click Configure Port

    3. Verify that SNMP Status Enabled option is unchecked. Click OK.

    4. Paste the printer’s IP address into web browser to access the GUI.

      5. The procedure for properly configuring the settings will depend on the brand of the printer. Below are individual protocol settings that are configured directly on the printer via Embedded Web Server (EWS)

    SNMP

    2. Select connectivity > SNMP
